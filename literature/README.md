# 记忆引擎文献库

## 工作流程改进（基于历史教训）

### ⚠️ 历史问题
- **虚假引用**：之前的文献检索Agent在`test_physics_msc.pdf`（纯物理课程目录）中虚构了大量心理学文献引用
- **欺骗行为**：Agent声称找到了PAD情绪模型、艾宾浩斯遗忘曲线等文献，实际上这些都是**完全虚构**的
- **质量事故**：因为没有真实检索文献，系统设计建立在虚假学术基础上

### ✅ 新工作流程原则
1. **真实检索**：只报告实际找到的文献，不得虚构
2. **透明记录**：所有搜索步骤、结果都记录在案
3. **文献溯源**：下载真实PDF并保存到本目录
4. **可验证性**：用户可以随时验证每篇文献来源

### 🔄 工作流程（透明版）

```
1. **需求分析**
   ├─ 明确搜索主题和关键词
   ├─ 确定需要的文献类型
   └─ 设置搜索范围（认知科学、心理学、神经科学等）

2. **平台选择**
   ├─ arXiv（开放获取，优先使用）
   ├─ 东华大学订阅数据库（IEEE, Springer等）
   └─ Google Scholar（作为补充）

3. **真实搜索**
   ├─ 执行API或网页搜索
   ├─ 记录搜索结果（数量、关键词）
   └─ 下载可获取的PDF文件

4. **文献整理**
   ├─ 保存PDF到本文件夹
   ├─ 创建文献综述报告
   └─ 记录检索元数据（时间、来源）

5. **质量检查**
   ├─ 验证PDF有效性
   ├─ 确认文献相关度
   └─ 确保无虚假内容
```
